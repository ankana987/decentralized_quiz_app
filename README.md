The Decentralized Quiz App is a blockchain-based application built on Stellar’s Soroban smart contract platform. It enables trustless, transparent, and tamper-proof quiz participation where questions, answers, and scores are stored on-chain.


Unlike traditional quiz systems, this platform ensures fairness, eliminates manipulation, and provides verifiable results using decentralized infrastructure.

---

## ⚙️ What it does

- Allows admins to create quiz questions on-chain
- Enables users to submit answers securely
- Automatically evaluates answers via smart contract logic
- Stores user scores permanently on blockchain
- Provides transparent and verifiable results

---

## 🚀 Features

- 🔐 Decentralized & trustless system
- 📊 Transparent scoring mechanism
- 🧾 On-chain data storage (quizzes & results)
- 👤 User-based score tracking
- ⚡ Fast and low-cost transactions (Stellar network)
- 🧩 Extensible for:
  - Multiple questions per quiz
  - Leaderboards
  - Token rewards (future scope)
